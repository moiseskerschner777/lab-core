# Phase 2 — AST Chunker

**Goal:** `docker compose run --rm app python -c "from app.chunker import chunk_codebase; ..."` extracts all chunk types from a mounted Python codebase — no IRIS or Ollama needed.

**Flow:** Flow 1 — Build→Observe→Adjust

**References:** `design.md` → `chunker.py`, `Chunk` dataclass

- [ ] 2.1 Create `app/chunker.py` with the `Chunk` dataclass and `chunk_id` helper.

  ```python
  import hashlib
  from dataclasses import dataclass

  @dataclass
  class Chunk:
      id: str
      file: str
      type: str        # "function" | "class" | "module" | "imports"
      name: str
      start_line: int
      end_line: int
      text: str
      module: str

  def chunk_id(file: str, type: str, name: str) -> str:
      return hashlib.sha1(f"{file}::{type}::{name}".encode()).hexdigest()
  ```

  **Test:**
  - Do: `docker compose run --rm app python -c "from app.chunker import chunk_id; print(chunk_id('a.py','function','foo'))"`
  - Expect: 40-char hex string. Same result on second run.
